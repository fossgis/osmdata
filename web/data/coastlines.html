Coastlines

Linestrings for coastlines. Long linestrings are split into smaller chunks (with no more than 100 points) that are easier and faster to work with.

Description

The data has been derived from OpenStreetMap ways tagged with natural=coastline. Some errors in the OSM data are repaired in the process. Learn more about the coastline processing...

This dataset can be used together with the land or water polygons.

Variants

  • The data is available in WGS84 and Mercator projection.
