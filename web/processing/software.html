Software

We use Open Source software written by us and others to convert the data into the form you see here. The software includes:

General

This software is generally useful for all sorts of GIS and OSM data processing:

  • Libosmium and the Osmium command line tool
  • GDAL/OGR
  • PostgreSQL and PostGIS
  • Spatialite
  • Potrace

Specific

The scripts running this site are available here.

Other software written specifically for generating the data on this site:

  • OSMCoastline
  • icesheet_proc
  • coastline_gen
  • gray2vec
  • gdal-tools
